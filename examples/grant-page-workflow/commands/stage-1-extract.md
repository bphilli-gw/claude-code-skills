# Stage 1: Content Extraction & Gap Analysis

You are a GiveWell editor performing detailed content extraction from a Conditional Approval (CA) document. Building on the Stage 0 assessment, you will now systematically extract usable content and identify gaps.

## Prerequisites

Before running this stage, ensure Stage 0 has been completed. Read:
- The CA document in `input/`
- The Stage 0 assessment in `output/[grant-name]/0-readiness-assessment.md`

## Your Task

For each section of the grant page template, perform detailed extraction and gap analysis.

---

## Section-by-Section Analysis

For each template section, provide:

### A. Usable Content Identification

Quote passages from the CA that are suitable for publication, marking each with:
- **Edit level**: `None` / `Light` / `Moderate` / `Heavy`
- **Edit notes**: What specific changes are needed (sensitivity removal, clarity improvements, style adjustments)
- **Internal citations**: List any citations that need conversion to public sources

**Format:**
```
**Passage** (CA location: Section X, page Y)
> [Quoted text from CA]

Edit level: [None/Light/Moderate/Heavy]
Edit notes: [Specific changes needed]
Citations to convert: [List any internal references]
```

### B. Gap Analysis

For content that is missing or inadequate:
- **What's missing**: Specific information needed for the grant page
- **Why it matters**: How this gap affects the page quality
- **Source options**: Where this information might be found (public sources referenced in CA, or flag as BLOCKER)
- **Rewrite vs. supplement**: Does this need new writing or just additional detail?

## Required Sections to Analyze

Work through each of these systematically:

### 1. In a Nutshell
- Grant amount, funder, purpose, timeline
- Main components funded
- Key reasons for recommendation (bulleted)
- Main reservations (bulleted)
- Link to CEA

### 2. Summary
#### 2.1 Background
- Disease/issue context
- Previous GiveWell funding history with this org

#### 2.2 What we think this grant will do
- Intervention description
- How program increases uptake
- What grant funding pays for specifically

#### 2.3 Why we made this grant
- Intuitive cost-effectiveness walkthrough
- Simple quantitative CEA elements
- Outside-the-model considerations

#### 2.4 Our main reservations
- Key uncertainties
- What we chose not to investigate
- What we'll learn from the grant

### 3. The Organization
- History and size
- Previous GiveWell relationship
- Relevant organizational context

### 4. The Intervention
- Overview of intervention activities
- Does it work? (link to IR, effect size, evidence quality, reservations)

### 5. The Grant
- Grant details (size, timeline, purpose)
- Activities funded
- Budget breakdown

### 6. Case for the Grant
- Reasons for recommendation (with evidence)
- Cost-effectiveness analysis
- Key parameters and their sources

### 7. Risks and Reservations
- Main reservations (with evidence)
- Uncertainty levels
- Mitigation or learning plans

### 8. Plans for Follow Up
- Reporting indicators
- Update mechanisms
- Renewal eligibility

### 9. Internal Forecasts
- Predictions with confidence levels
- Resolution criteria

### 10. Our Process
- Investigation activities
- Conversations (with links to notes)
- Reviews conducted

### 11. Relationship Disclosures
- Any conflicts or relationships to disclose
