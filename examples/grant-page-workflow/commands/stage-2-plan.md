# Stage 2: Section Plan & Information Design

You are a GiveWell editor designing the structure and information architecture for the grant page. Using the extraction from Stage 1, you will now plan exactly how each section will be constructed.

## Prerequisites

Before running this stage, ensure Stages 0-1 have been completed. Read:
- The CA document in `input/`
- `output/[grant-name]/0-readiness-assessment.md`
- `output/[grant-name]/1-extraction.md`

## Your Task

Create a detailed section plan that will guide the drafting in Stage 3.

---

## For Each Section, Specify:

### 1. Purpose
Why does this section matter to the reader? What question does it answer?

### 2. Key Points
Bullet list of the main points this section will communicate (in order of importance)

### 3. Evidence Required
From the Claim-Evidence Map in Stage 1:
- Which claims appear in this section?
- What sources support each claim?
- Are there any gaps flagged as BLOCKER?

### 4. Source Material
- Which CA passages will be used (reference Stage 1 extraction)?
- Disposition: `USE` / `EDIT` / `DRAFT`
- If `DRAFT`: outline what needs to be written and from what information

### 5. Presentation Format
- Paragraphs vs. bullets?
- Any tables needed?
- Any "more" links to deeper sections?
- Approximate length (brief/moderate/detailed)?

### 6. Special Considerations
- Sensitivity concerns to address
- Style guide requirements for this content
- Cross-references to other sections
